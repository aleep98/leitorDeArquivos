# Leitor de Arquivos

Projeto simples para leitura e processamento de arquivos de texto.
Ele lê conteúdos de arquivos na pasta `arquivos/`, trata erros e gera resultados em `resultados/`.

## Para que serve
- Extrai o texto dos arquivos de entrada
- Processa e valida o conteúdo
- Gera um arquivo de saída com o resultado do processamento

## Como usar
1. Coloque arquivos `.txt` na pasta `arquivos/`
2. Execute o script principal:
   `node src/index.js`
3. Verifique o resultado em `resultados/resultado.txt`

## Requisitos
- Node.js instalado
- Dependências do projeto instaladas via `npm install`
